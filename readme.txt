== Description ==

A simple widget for showing responsive image in sidebar area. It's using WordPress's new media uploader.


> [DEMO](http://demo1.wpbean.com/)  |  [Support](https://wpbean.com/support/)

### Plugin Features

* Showing image in sidebar by widget.
* Linking image to large image or cutom link.
* Image size.
* Image alt text.
* Image bellow text.
* Image alignment.
* Nice animation on mouse hover.
* Easy to use.
* Clean code & easy to customize.

== Installation ==

* Install it as a regular WordPress plugin

* Active the plugin.

* Navigate to Appearance > Widgets. You will find this plugin's widegt called "WPB Image Widget".

* Drap it to your widget area and configure it's settings.



== Frequently asked questions ==

= How to change the image size? =
In this plugin's widget options you will see a dropdown with all sizes defined in your WordPress site. Select any size form here.

= How can I change link target of image linking? =
In this plugin's widget options you will see a dropdown called "Link Window" for this.
